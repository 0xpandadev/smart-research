# Forecasting Lenses

Use when the user asks what may happen next, how a market may evolve, whether a trend will continue, or how to think about future scenarios.

## Forecast Structure

Frame forecasts as:

- question
- time horizon
- base rate
- key drivers
- leading indicators
- scenarios
- probability or confidence range
- disconfirming evidence
- update triggers

## Forecasting Discipline

- Start with base rates before narrative.
- Separate what is already true from what must become true.
- Identify bottlenecks and constraints.
- Track leading indicators, not only lagging outcomes.
- Prefer ranges and scenarios over false precision.

## Scenario Template

Use three scenarios when helpful:

- Base case: most likely path if current evidence continues.
- Upside case: what must improve or accelerate.
- Downside case: what breaks, slows, or reverses.

For each scenario, include:

- evidence today
- required assumptions
- leading indicators
- what would falsify it

## Output Checks

- Do not present a forecast as a fact.
- Include timeframe.
- Include uncertainty.
- Name the indicators that would make you change your mind.
