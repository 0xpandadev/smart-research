# Deep Reading Playbook

Use when sources include long reports, PDFs, decks, tables, diagrams, images, or visual evidence.

## Document Reading

Read for:

- Thesis and conclusion
- Methodology and sample
- Metrics, denominators, units, time period, geography
- Footnotes and caveats
- Tables and chart values
- Visual structures: 2x2 matrices, market maps, stacks, funnels, value chains, architecture diagrams
- What is missing or selectively framed

## Visual Reading

For charts, screenshots, slides, and diagrams:

- Identify the visual type and intended message.
- Extract axes, labels, legends, units, categories, and ranked elements.
- Translate diagrams into structured text: nodes, relationships, layers, flows, dependencies.
- Mark inferred visual meaning separately from text explicitly present in the image.
- Note low confidence when resolution, cropping, or OCR quality limits reading.

## Slide/Consulting Decks

Look for:

- Headline claim on each slide
- Supporting exhibit
- Data source and footnotes
- Framework or segmentation logic
- Implication for the decision
- Hidden assumptions behind the framing

## Output

When deep reading matters, include a compact table:

`Source | Page/Section | Evidence | Interpretation | Confidence | Follow-up`
