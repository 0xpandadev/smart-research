# Source Policy

Never rely on one provider, one ranking surface, or one type of source.

## Reliability Tiers

- Tier 1: Official primary sources, filings, annual reports, investor decks, regulatory agencies, government statistics, standards bodies, court/regulatory documents, product docs, source repos.
- Tier 2: Reputable specialist media, trade publications, analyst/consulting reports, academic papers, industry associations, conference talks, expert interviews.
- Tier 3: Company blogs, technical blogs, GitHub issues, changelogs, vendor docs, implementation notes, customer case studies.
- Tier 4: Reddit, X, HN, forums, reviews, comments, newsletters, podcasts, YouTube captions, informal operator posts.

Use Tier 4 for weak signals, friction, complaints, adoption clues, language customers use, and emerging behavior. Do not use Tier 4 alone for factual conclusions.

## Source Diversity Rule

- Use search engines for discovery, not final authority.
- Anchor high-stakes facts in primary sources whenever possible.
- Use direct source navigation for likely primary sources: company IR, SEC/EDINET, regulatory pages, standards bodies, official docs, GitHub, papers.
- Run contradiction searches: criticism, lawsuit, failure, churn, outage, security issue, regulation, competitor, "not working", "alternative", "problem".
- Track dates. Prefer recent sources for volatile topics, and original sources for historical facts.

## Credibility Checks

For each important source, assess:

- Who produced it?
- What incentive might bias it?
- Is it primary, secondary, or commentary?
- What date and geography does it cover?
- Does another independent source support it?
- What would disprove it?

## Original Evidence Rule

For important claims, preserve enough original evidence for the user to inspect:

- Short original excerpt
- Table value with unit and period
- Page, section, slide, table, figure, filing item, or document title when available
- Chart or image description when visual evidence matters
- Source date and publisher

Do not quote long copyrighted passages. Use short excerpts and paraphrase the rest.

## Source Citation Rule

Do not make important factual claims without a source. Prefer primary or near-primary sources for core facts. If only secondary or weak-signal sources are available, label the confidence accordingly.

If page, slide, section, or table references are unavailable, say so plainly instead of inventing a location. Useful citations are inspectable, not decorative.
